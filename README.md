📊 Revenue Operations Dashboard - Portfolio Project
This repository contains a business performance dashboard built using Streamlit and SQLite. The project focuses on data analysis, visualization, 
and performance tracking for key business metrics like revenue growth, sales performance, customer acquisition, and marketing efficiency.

🚀 Project Overview
This project aims to show:
  - An understanding of RevOps – The key areas of concern, the metrics, KPIs, and measures that define success across sales, marketing, customer success.
  - How to structure and analyze business data – Organizing revenue, sales performance, customer acquisition, and marketing data into meaningful insights.
  - Data analysis using SQL and Pandas – Extracting, transforming, and interrogating data for actionable intelligence.
  - Visualization of key business metrics 

📌 Key Sections in the Dashboard

  1. Revenue Overview - MRR/ARR trends, revenue breakdown.
  2. Sales Performance - Pipeline analysis, win rates, sales cycle.
  3. Customer Acquisition & Retention - CAC, CLTV, churn rates.
  4. Marketing Performance - Campaign performance, MQL/SQL conversion.
  5. Customer Success - NPS scores, expansion revenue.
